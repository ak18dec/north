# North

> Learn what matters.

AI-powered personalized learning recommendations that help engineers discover what’s worth learning next.

North helps cut through endless technical content and surface learning opportunities aligned with your skills, interests, and career direction.

---

## Why North?

Technology evolves faster than engineers can realistically keep up.

Every day brings:

* Hacker News
* GitHub trends
* Blogs
* Reddit discussions
* New frameworks
* Endless recommendations

Most content is interesting.

Very little is worth your time.

North helps answer:

**What should I learn next?**

```bash

```

Run locally:

```bash

```

Create production build:


